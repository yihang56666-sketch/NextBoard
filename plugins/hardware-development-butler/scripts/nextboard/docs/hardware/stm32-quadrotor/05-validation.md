# STM32 小型四旋翼无人机验证计划

**Project:** STM32 Small Quadrotor Drone
**Stage:** 05 - Validation
**Date:** 2026-06-01

## EVT

- 检查 MCU 上电和 SWD 连接
- 验证 IMU 读数稳定
- 验证 UART 日志输出
- 验证接收机输入解码

## 台架测试

- 单电机方向测试
- 四电机混控测试
- 解锁/上锁测试
- 失控保护测试

## 无桨测试

- 姿态响应测试
- PID 参数粗调
- 电源纹波检查
- 温升检查

## 上桨短悬停

- 低油门起飞
- 轻微扰动恢复
- 低电压保护
- 紧急断油

## 通过标准

- 传感器无明显异常跳变
- 控制环无持续振荡
- 电源不掉压到危险区
- 保护逻辑可以稳定触发
