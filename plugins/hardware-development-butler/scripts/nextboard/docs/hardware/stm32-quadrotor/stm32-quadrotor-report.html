目标

做一台可实验、可迭代、可悬停的小型四旋翼原型，优先验证飞控链路、姿态控制、电源和安全策略。

关键约束

  • 机型：75 mm 到 3 寸级实验平台
  • 控制器：STM32 G4 系列，优先 STM32G431
  • 执行链路：外置 4-in-1 ESC + 无刷电机
  • 传感器：SPI 六轴 IMU，优先高输出数据率器件
  • 供电：2S 或 3S 锂电
  • 调试：SWD + 串口日志
  • 第一版不做：自研电调、高级避障、图传、云台

核心功能

  • 姿态估计
  • 三轴角速度闭环
  • 电机混控
  • 解锁/上锁
  • 低电压保护
  • 失控/掉线保护
  • 日志和地面调参接口

成功标准

  • 地面台架能稳定识别姿态
  • 四电机方向和响应一致
  • 无桨测试下 PID 可调通
  • 上桨后可完成短时悬停实验

已知假设

  • 电机和 ESC 采用成熟现成方案
  • 先用飞控板 + 电调分离架构
  • 先追求稳定和可维护，不追求极限续航

STM32 小型四旋翼无人机架构

Project: STM32 Small Quadrotor Drone
Stage: 02 - Architecture
Date: 2026-06-01

总体方案

推荐采用“STM32 飞控板 + 外置 4-in-1 ESC + 轻量机架”的分体式架构。这样能把高速功率部分与控制部分分开，降低第一版风险，也方便后续换机架和换电机。

方案选择

推荐方案 A

STM32G431 + SPI IMU + 4-in-1 ESC + 2S/3S 电池

  • 优点：实时性好，控制资源足，适合飞控
  • 缺点：需要更认真做 PCB 和供电隔离

备选方案 B

STM32F405/F411 + SPI IMU + 4-in-1 ESC + 2S/3S 电池

  • 优点：生态成熟
  • 缺点：控制资源和模拟外设不如 G4 贴近电机控制需求

不推荐方案 C

STM32 + 自研四路无刷驱动

  • 优点：集成度高
  • 缺点：第一版风险太大，调试成本高

模块划分

  • 飞控主控：STM32G431
  • 姿态传感：ICM-42688-P 级 SPI IMU
  • 高度辅助：BMP390 级气压计，可选
  • 供电：VBAT -> 5V -> 3.3V
  • 通信：USB 或 UART 调试口，预留接收机输入
  • 执行：4 路 ESC 输出

数据流

IMU -> 姿态估计 -> PID -> 混控 -> ESC -> 电机

风险点

  • IMU 抖动和振动耦合
  • 电源噪声导致传感器读数不稳
  • 板级布局不当导致 USB/SWD 调试不稳定
  • 首版参数调得过激导致炸机

STM32 小型四旋翼无人机器件建议

Project: STM32 Small Quadrotor Drone
Stage: 03 - Components
Date: 2026-06-01

主控

推荐：STM32G431

  • 理由：G4 系列面向电机控制，资源更适合飞控实时任务
  • 用途：主控制、PID、串口调参、状态机、混控
  • 选择原则：优先拿到官方数据手册确认封装、Flash、RAM、定时器和 ADC 资源

姿态传感器

推荐：ICM-42688-P

  • 理由：SPI 接口、适合高频姿态采样
  • 用途：陀螺仪与加速度计主传感器
  • 选择原则：优先官方数据手册，确认量程、输出速率、噪声和工作电压

辅助传感器

可选：BMP390

  • 理由：用于低速高度辅助
  • 用途：起降和悬停辅助
  • 选择原则：如果第一版预算紧，可先不装

通信与调试

  • SWD 调试口
  • UART 日志口
  • 接收机输入：SBUS 或 CRSF 任选其一

执行链路

  • 4-in-1 ESC
  • 4 个无刷电机
  • 推荐先用成熟电机尺寸和 KV，不自研驱动

电源

  • 电池：2S 或 3S LiPo
  • 主电源：VBAT
  • 降压：VBAT -> 5V -> 3.3V
  • 传感器与 MCU 统一 3.3V 域，模拟/数字去耦分开

先验结论

  • 第一版最关键的是 IMU、供电和布局，不是芯片堆料
  • 不要把预算花在复杂外设上，先把飞稳做出来

STM32 小型四旋翼无人机约束

Project: STM32 Small Quadrotor Drone
Stage: 04 - Constraints
Date: 2026-06-01

接口约束

  • IMU 走 SPI，不建议第一版用 I2C 主姿态链路
  • SWD 必须保留
  • UART 调试口必须保留
  • 4 路电调输出必须独立

电源约束

  • 3.3V 数字电源要干净
  • IMU 和 MCU 供电去耦要靠近器件
  • 5V 与 VBAT 侧要注意大电流回路
  • 外部电机电流不要穿过控制地

PCB 约束

  • IMU 靠近板中心放置
  • 尽量让主轴对称
  • 远离电源开关节点和电机大电流走线
  • 高速走线短而直
  • 4 层板更稳妥

机械约束

  • 首版优先小机架
  • 飞控板要能装进常见 75 mm 到 3 寸机架
  • 重心尽量贴近机体中心

走线宽度保守值

按 10°C 温升、外层、1 oz 铜估算：

  • 0.5 A 级信号/小电源：约 4.47 mil
  • 2 A 级电源：约 30.28 mil

因此主电源走线、地回流和 ESC 供电应明显宽于普通信号线。

STM32 小型四旋翼无人机原理图连接表

Project: STM32 Small Quadrotor Drone
Stage: 07 - Schematic Link Map

模块连接

STM32G431

  • VDD -> 3.3V
  • VSS -> GND
  • NRST -> 复位电路 + 调试口
  • SWDIO/SWCLK -> 调试口
  • SPI1 -> IMU
  • UART1 -> 日志口
  • UART2 -> 接收机或地面站
  • TIM -> 4 路电调输出

ICM-42688-P

  • VDD -> 3.3V
  • GND -> GND
  • SCK/MISO/MOSI/CS -> STM32 SPI
  • INT1 -> 中断输入

BMP390（可选）

  • VDD -> 3.3V
  • GND -> GND
  • SCK/MISO/MOSI/CS -> 共享 SPI 或独立 SPI

电源

  • 电池输入 -> 保护/开关 -> 5V 降压
  • 5V -> 3.3V LDO/DC-DC
  • 3.3V -> MCU + 传感器

执行器

  • MOTOR1..4 -> 4-in-1 ESC 信号

布局优先级

  1. IMU 放板中心
  2. 电源和开关节点远离 IMU
  3. SWD 口靠板边
  4. 传感器和 MCU 的去耦电容贴近引脚

1. 项目目标

做一台可实验、可迭代、可悬停的小型四旋翼原型，优先验证飞控链路、姿态控制、电源和安全策略。

2. 核心约束

  • 机型：75 mm 到 3 寸级实验平台
  • 主控：STM32 G4 系列，优先 STM32G431
  • 执行链路：外置 4-in-1 ESC + 无刷电机
  • 传感器：SPI 六轴 IMU
  • 供电：2S 或 3S 锂电
  • 调试：SWD + 串口日志
  • 第一版不做：自研电调、高级避障、图传、云台

3. 推荐架构

推荐采用“STM32 飞控板 + 外置 4-in-1 ESC + 轻量机架”的分体式架构。

数据流：

IMU -> 姿态估计 -> PID -> 混控 -> ESC -> 电机

这种架构把高速功率部分与控制部分分开，降低第一版风险，也方便后续换机架、换电机和调试控制算法。

4. 关键器件

模块 推荐器件/方案 说明
MCU STM32G431 用于主控制、PID、混控、接口管理
IMU ICM-42688-P 级 SPI IMU 用于姿态估计主传感
高度辅助 BMP390 级气压计 可选，第一版可不装
电调 成品 4-in-1 ESC 降低功率级调试风险
电源 VBAT -> 5V -> 3.3V MCU 和传感器统一 3.3V 域

5. 电源树

2S/3S LiPo
  -> VBAT 输入保护
  -> 5V 降压
  -> 3.3V 稳压
  -> STM32 + IMU + 通信接口

电机大电流路径不要穿过控制地。IMU 与 MCU 的去耦电容必须靠近引脚。

6. 接口规划

接口 用途 约束
SPI IMU 主链路 不建议第一版用 I2C 做主姿态链路
SWD 下载和调试 必须保留
UART 日志/接收机/地面站 至少保留 2 路
TIM 输出 4 路 ESC 信号 独立输出

7. PCB 约束

  • IMU 靠近板中心
  • 主轴尽量对称
  • IMU 远离电源开关节点和电机大电流走线
  • 传感器和 MCU 去耦电容贴近引脚
  • 4 层板优先

走线宽度参考：

  • 0.5 A 级小电源：约 4.47 mil
  • 2 A 级电源：约 30.28 mil
