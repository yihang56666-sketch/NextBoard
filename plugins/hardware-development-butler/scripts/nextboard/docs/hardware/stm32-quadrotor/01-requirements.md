# STM32 小型四旋翼无人机需求定义

**Project:** STM32 Small Quadrotor Drone
**Stage:** 01 - Requirements
**Date:** 2026-06-01

## 目标

做一台可实验、可迭代、可悬停的小型四旋翼原型，优先验证飞控链路、姿态控制、电源和安全策略。

## 关键约束

- 机型：75 mm 到 3 寸级实验平台
- 控制器：STM32 G4 系列，优先 STM32G431
- 执行链路：外置 4-in-1 ESC + 无刷电机
- 传感器：SPI 六轴 IMU，优先高输出数据率器件
- 供电：2S 或 3S 锂电
- 调试：SWD + 串口日志
- 第一版不做：自研电调、高级避障、图传、云台

## 核心功能

- 姿态估计
- 三轴角速度闭环
- 电机混控
- 解锁/上锁
- 低电压保护
- 失控/掉线保护
- 日志和地面调参接口

## 成功标准

- 地面台架能稳定识别姿态
- 四电机方向和响应一致
- 无桨测试下 PID 可调通
- 上桨后可完成短时悬停实验

## 已知假设

- 电机和 ESC 采用成熟现成方案
- 先用飞控板 + 电调分离架构
- 先追求稳定和可维护，不追求极限续航
