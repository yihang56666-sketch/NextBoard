# STM32 小型四旋翼无人机方案汇总

**Project:** STM32 Small Quadrotor Drone
**Version:** 0.1
**Date:** 2026-06-01

## 方案概述

这是一台面向实验和迭代的 STM32 小型四旋翼无人机原型，采用 STM32G431 作为飞控主控，配合 SPI IMU、外置 4-in-1 ESC 和 2S/3S 锂电池，优先完成姿态稳定、混控、保护和短时悬停验证。

## 关键选择

- 主控：STM32G431
- 姿态传感：ICM-42688-P
- 高度辅助：BMP390，可选
- 执行链路：成熟 4-in-1 ESC + 无刷电机
- 供电：VBAT -> 5V -> 3.3V

## 风险

- 供电噪声
- 振动耦合
- 首次调参过激
- 机身重心与 IMU 布局不合理
