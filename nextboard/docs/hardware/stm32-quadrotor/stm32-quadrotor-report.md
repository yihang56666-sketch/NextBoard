# STM32 小型四旋翼无人机硬件方案报告

**Project:** STM32 Small Quadrotor Drone
**Version:** 0.1
**Date:** 2026-06-01

## 1. 项目目标

做一台可实验、可迭代、可悬停的小型四旋翼原型，优先验证飞控链路、姿态控制、电源和安全策略。

## 2. 核心约束

- 机型：75 mm 到 3 寸级实验平台
- 主控：STM32 G4 系列，优先 STM32G431
- 执行链路：外置 4-in-1 ESC + 无刷电机
- 传感器：SPI 六轴 IMU
- 供电：2S 或 3S 锂电
- 调试：SWD + 串口日志
- 第一版不做：自研电调、高级避障、图传、云台

## 3. 推荐架构

推荐采用“STM32 飞控板 + 外置 4-in-1 ESC + 轻量机架”的分体式架构。

数据流：

```text
IMU -> 姿态估计 -> PID -> 混控 -> ESC -> 电机
```

这种架构把高速功率部分与控制部分分开，降低第一版风险，也方便后续换机架、换电机和调试控制算法。

## 4. 关键器件

| 模块 | 推荐器件/方案 | 说明 |
|---|---|---|
| MCU | STM32G431 | 用于主控制、PID、混控、接口管理 |
| IMU | ICM-42688-P 级 SPI IMU | 用于姿态估计主传感 |
| 高度辅助 | BMP390 级气压计 | 可选，第一版可不装 |
| 电调 | 成品 4-in-1 ESC | 降低功率级调试风险 |
| 电源 | VBAT -> 5V -> 3.3V | MCU 和传感器统一 3.3V 域 |

## 5. 电源树

```text
2S/3S LiPo
  -> VBAT 输入保护
  -> 5V 降压
  -> 3.3V 稳压
  -> STM32 + IMU + 通信接口
```

电机大电流路径不要穿过控制地。IMU 与 MCU 的去耦电容必须靠近引脚。

## 6. 接口规划

| 接口 | 用途 | 约束 |
|---|---|---|
| SPI | IMU 主链路 | 不建议第一版用 I2C 做主姿态链路 |
| SWD | 下载和调试 | 必须保留 |
| UART | 日志/接收机/地面站 | 至少保留 2 路 |
| TIM 输出 | 4 路 ESC 信号 | 独立输出 |

## 7. PCB 约束

- IMU 靠近板中心
- 主轴尽量对称
- IMU 远离电源开关节点和电机大电流走线
- 传感器和 MCU 去耦电容贴近引脚
- 4 层板优先

走线宽度参考：

- 0.5 A 级小电源：约 4.47 mil
- 2 A 级电源：约 30.28 mil

## 8. 原理图连接摘要

### STM32G431

- VDD -> 3.3V
- VSS -> GND
- NRST -> 复位电路 + 调试口
- SWDIO/SWCLK -> 调试口
- SPI1 -> IMU
- UART1 -> 日志口
- UART2 -> 接收机或地面站
- TIM -> 4 路电调输出

### ICM-42688-P

- VDD -> 3.3V
- GND -> GND
- SCK/MISO/MOSI/CS -> STM32 SPI
- INT1 -> 中断输入

## 9. 验证计划

### EVT

- MCU 上电
- SWD 连接
- IMU 读数
- UART 日志
- 接收机输入

### 台架测试

- 单电机方向
- 四电机混控
- 解锁/上锁
- 失控保护

### 无桨测试

- 姿态响应
- PID 粗调
- 电源纹波
- 温升

### 上桨短悬停

- 低油门起飞
- 扰动恢复
- 低电压保护
- 紧急断油

## 10. 风险清单

| 风险 | 等级 | 缓解动作 |
|---|---|---|
| IMU 振动耦合 | 高 | IMU 居中，结构减振，无桨先看频谱 |
| 电源噪声 | 高 | 分区布局，去耦贴近，台架测纹波 |
| PID 过激 | 高 | 无桨调试，限幅，低油门首次起飞 |
| 首版板框不适配机架 | 中 | 先确认 75 mm 或 3 寸机架规格 |
| 接收机协议不确定 | 中 | 原理图预留 UART，固件支持 SBUS/CRSF 二选一 |

## 11. 已决策与待确认

已决策：

- 主控：STM32G431
- 架构：飞控板 + 外置 4-in-1 ESC
- 主姿态传感：SPI IMU
- 第一版不做自研三相功率级

待确认：

- 机架尺寸：75 mm 还是 3 寸
- 电池节数：2S 还是 3S
- 接收机协议：SBUS 还是 CRSF
- 是否加入气压计
