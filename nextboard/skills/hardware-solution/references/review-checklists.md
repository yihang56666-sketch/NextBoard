# Hardware Review Checklists

Use these lists when reviewing a hardware design, schematic, PCB constraint set, BOM, or design proposal.

## 原理图评审

- 主控启动配置、复位、时钟、下载调试接口是否完整。
- 每个电源引脚是否有合适去耦，电源域命名是否清晰。
- 电源芯片反馈、电感、电容、补偿、使能、PG、软启动是否符合数据手册。
- 输入口是否有反接、过压、浪涌、ESD、短路保护。
- 外部接口电平是否匹配，是否需要隔离或电平转换。
- I2C/SPI/UART/CAN/RS485 等总线的上拉、终端、电平、地址冲突是否明确。
- 模拟前端是否有输入保护、滤波、基准源、地划分和校准路径。
- USB/Ethernet/RF/高速接口是否按参考设计保留阻抗、ESD、共模电感和连接器约束。
- 电池和充电路径是否覆盖温度检测、保护、运输模式、认证要求。
- 测试点是否覆盖电源轨、关键总线、复位、启动模式、烧录和量产校准。

## PCB 评审

- 电源大电流路径短且宽，回流路径连续。
- DCDC 热回路、开关节点、反馈路径按数据手册布局。
- 模拟、数字、射频、功率区域有清晰边界，避免关键噪声耦合。
- 高速差分线等长、阻抗、参考平面、过孔数量符合要求。
- 晶振靠近芯片，负载电容、地护环和远离干扰源处理合理。
- 天线净空、匹配网络、接地过孔、金属结构距离符合射频建议。
- 连接器受力、ESD 泄放路径、安装孔接地和边缘距离合理。
- 测试点可接触，不被结构遮挡，满足夹具方向和针床间距。
- 热源有铜皮、过孔、散热路径和温升验证计划。

## BOM 评审

- 关键芯片至少有生命周期、供货周期、第二来源或替代策略。
- 阻容感封装不过度小型化，符合制造能力和良率目标。
- 高精度、低噪声、高压、大电流器件参数有余量。
- 连接器、线束、电池、屏幕等长周期物料已提前锁定。
- 无许可证、停产、独家代理、区域受限或认证不清的器件被标为风险。
- 物料等级符合环境要求：工业级、车规、医疗或消费级边界明确。

## 设计方案评审

- 是否解释了“为什么是这个架构”，而不仅是“用了哪些器件”。
- 成本、功耗、可靠性、软件复杂度之间的取舍是否可接受。
- 是否列出待确认项，并说明谁负责确认。
- 是否存在依赖单一供应商、单一工具链、单一认证路径的风险。
- 是否有从 EVT 到 PVT 的验证闭环。
