# Download Sources Registry

下载经验记录。每次成功下载 datasheet 或封装文件后，将器件型号、成功源站点和 URL 记录到此文件。下次遇到相同或同系列器件时，优先使用已验证的源。

## 使用规则

1. 下载前先查阅本文件，如果目标器件或同系列器件已有记录，直接使用已验证的源
2. 下载成功后，将新记录追加到对应分类下
3. 如果已记录的源失效，标注失效日期并补充新的可用源
4. 记录格式：`| 器件型号 | 源站点 | URL | 验证日期 |`
